## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/E-commerce-wasp.git
   cd E-commerce-wasp
   ```
2.  Install frontend & backend dependencies:

   ```bash
   wasp db migrate-dev # migrate database
   ```

3. Run the project:

   ```bash
   wasp start # Enjoy :)
   ```
